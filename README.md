# 学生管理系统简介 java998

## 项目概述

学生管理系统是一款基于SSM框架的后台管理系统，主要用于对学生信息的管理。代码注释详细，逻辑结构清晰，适合初学者参考和学习。

- IntelliJ IDEA
- MySQL

### 技术选型
- JSP
- jQuery
- Ajax
- EasyUI
- Spring
- Spring MVC
- MyBatis
- MySQL
- Maven

## 功能模块

- 学生信息管理
  - 查询
  - 添加
- 教师信息管理
  - 查询
  - 添加

## 系统角色

- 管理员：具有所有管理模块的操控权限。
- 学生：仅具有学生信息管理模块的查询及添加信息的权限。
- 教师：具有学生信息管理模块的所有权限，且在教师信息管理模块中只具有查询及添加信息的权限。
